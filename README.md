# JsonB64 - Handler Text and Json formatter

> [!TIP]
> For the JSON language editor, [CodeMirror](https://codemirror.net/) was used

### Development Setup

First, clone the repo and cd into the project:

## Local setup

Install dependencies

```bash
npm i
```

Run angular/cli command

```bash
npm run ng
```

## Run locally

Development server

```bash
npm run dev
```

Generate build

```bash
npm run build
```
